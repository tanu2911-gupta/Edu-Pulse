# EduPulse 🎓📊

EduPulse is an AI-powered student analytics and academic management platform designed to improve student performance tracking and educational insights.

## 🚀 Features

* Attendance Tracking
* Internal Marks Analysis
* Student Dashboard
* AI-based Result Prediction
* Graphical Data Visualization
* Authentication System
* Responsive UI

## 🛠️ Technologies Used

* React.js
* JavaScript
* CSS
* Axios
* React Icons

## 📌 Purpose

The project helps educational institutions analyze student performance and manage academic data efficiently through interactive dashboards and AI-powered insights.
